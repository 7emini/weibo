# Weibo - Laravel 6 练手项目

[教程地址](https://learnku.com/courses/laravel-essential-training/6.x)

[阿里云部署地址](http://wb.7emini.club/)

## 技术栈

* 后端：`Laravel 6.12.0`
* 前端使用 `Bootstrap4`
* 数据库使用 `MySQL`
* 版本控制使用 `Git`

## 学习技能

* 部署项目上线
* 项目配置
* `git` 工作流程
* `artisan` 命令
* 请求和响应
* `Eloquent`模型操作
* 注册登录
* 权限控制
* 提交数据验证
* 中间件
* Blade模版
* 发送邮件
